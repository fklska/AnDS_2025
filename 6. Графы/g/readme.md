# Задача G. Offline-семинар
### У графа есть понятие эксцентриситета - по простому это "центр графа"
### По хорошему это такая вершина(ы) максмиальное расстояние до всех других вершин минамально
### Так как граф взвешенный, задача решается через алгоритм дейкстры для каждого нода
### 1. Нужно найти маскимальное расстояние от конкретного нода до любого другого
### 2. Среди всех максимальных расстояний выбрать минимальное - это и будет эксцентриситет
### Реализацию дейкстры можно посмотреть в инете
### Но вкраце: 
Нужно записывать минимальные рассотяния из нода в нод, выбрать максимальное, так для каждого нода и среди максимальных выбрать минимальное
