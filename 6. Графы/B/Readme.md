# Задача B. Есть ли цикл?
## O(n)
## Читаем ориентированный граф
### Запускаем DFS
### При заходе в ноду помечаем ее серым цветом
### При выходе черным
### Идея такая мы не должны 2 раза попадать в одну и туже (черную ноду) из разных DFS
### Как пример 3 тест, зачем нужен серый цвет
### Так как граф ориентированный то в из одной ноды в другую ноду можно попасть разными путями, а обратно не факт
### И поэтому цикла нет
